# Pomodoro Timer

A simple Pomodoro timer built with Python and Tkinter. This desktop application helps you focus by breaking your work into intervals (work and breaks) following the Pomodoro Technique.

---

##  Features

- 25-minute work sessions
- 5-minute short breaks
- 20-minute long break every 4 work sessions
- Visual countdown timer with start/reset buttons with clean UI
- Check mark display for completed work intervals

---

## 💡 Usage

- Click **START** to begin a work session.
- The timer will automatically switch between work and break intervals.
- Click **RESET** to stop the timer and clear progress.

---

##  Customization

You can adjust the constants at the top of `main.py`:

```python
WORK_MIN = 25
SHORT_BREAK_MIN = 5
LONG_BREAK_MIN = 20
```

Modify these to change the duration of sessions.
